Overview: Building an MVP with Cursor IDE

  • Generate customized rules with Cursor Rules GPT to guide your build process
  • Build your MVP one feature at a time for modular development

Step 0: Prerequisites (Initial Setup)

  • Sign up for Cursor IDE and configure your development environment
  • Download and install Cursor - available for Windows, macOS, and Linux
  • Alternative: Install VS Code if using Roo-Code extension for team collaboration

Step 1: Generate Cursor Rules with GPT

  • Access Cursor Rules GPT to generate your custom .cursorrules file
  • Input your tech stack (Frontend, Backend, Database) for tailored rules
  • Download the .cursorrules file for Cursor IDE integration

Step 2: Feed Master Plan & Rules into Cursor
